- 🛡️ Secure Boot (with custom keys)
- 🧽 Impermanence
- 🔐 Full disk encryption
- 🥸 Secrets management for both [evaluation](./docs/src/evalSecrets.md) and activation time
- 🎨 Stylix colorscheme
- 🗃️ Documentation
